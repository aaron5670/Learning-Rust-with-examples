# Personal project to learn Rust

## Pre-requirements
1. Download and install Rust [here](https://www.rust-lang.org/tools/install).

### Hello world app in Rust
1. Open a terminal.
2. ``$ cd hello_cargo``
3. ``$ cargo run``


### Guessing game
1. Open a terminal.
2. ``$ cd guessing_game``
3. ``$ cargo run``


### MongoDB example
1. Open a terminal.
2. ``$ cd rust_mongodb_example``
3. ``$ cargo run``
